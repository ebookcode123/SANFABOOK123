题目描述

请计算一个二叉树的平均宽度

例如，


     0
    / \
   1   2
  / \   \
 3  4    5

这个树一共三层，每一层的宽度为：1、2、3，因此平均宽度为2。

输入输出

  • 第一行是n，表示二叉树的节点个数；当n=-1时，表示输入结束

  • 之后的n行，每行是每个节点的左子树和右子树的节点编号；编号-1表示空

输出

  • 针对每组数据，输出这个二叉树的平均宽度（将结果取整）

示例输入

6
1 2
3 4
0 5
-1 -1
-1 -1
-1 -1
-1

示例输出

2

题目描述

排序两个二叉搜索树的元素并输出

特殊要求：

  • 请使用以下数据结构 
class Node
{
    int value;
    Node left;
    Node right
    }
  • 额外使用O(1)空间

输入输出

  • 第一行是n，表示第一个二叉搜索树的节点个数；当n=-1时，表示输入结束

  • 之后的n行，每行是每个节点的左子树和右子树的节点编号，以及这个节点保存的值；编号-1表示空

  • 第n+2行是m，表示第二个二叉搜索树的节点个数

  • 之后的m行，每行是每个节点的左子树和右子树的节点编号，以及这个节点保存的值；编号-1表示空

输出

  • 针对每组数据，输出将两个二叉搜索树的元素排序后的结果

示例输入

3
1 2 10
-1 -1 1
-1 -1 20
3
1 2 15
-1 -1 2
-1 -1 40
-1

示例输出

1 2 10 15 20 40

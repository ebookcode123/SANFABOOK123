题目描述

请将一个环形链表拆分为两个等长的环形链表；如果环形链表长度为奇数，则抛弃最中间的元素

特殊要求

  • 请使用以下链表结构 
class Node
    {
    int data;
    Node* next;
    };

输入输出

输入

  • 第一行是n，表示节点个数；当n=-1，表示输入结束

  • 第二行是n个整数，表示链表内保存的元素

输出

  • 针对每组数据，第一行输出拆分后的第一个链表的元素，第二行输出拆分后的第二个链表的元素

示例输入

4

1 2 3 4

5

1 2 1 2 1

-1

示例输出

1 2

3 4

1 2

2 1

题目描述

对有n个元素的堆栈排序

  • 请使用以下堆栈结构 
class Stack
    {
    int push();
    int pop();
    }
  • 请额外使用O(1)的空间

输入输出

输入

  • 第一行是n；当n=-1时，表示输入结束

  • 第二行是依次放入栈内的元素

输出

  • 针对每组数据，输出排好序的栈的数据

示例输入

4

5 2 1 4

-1

示例输出

1 2 4 5

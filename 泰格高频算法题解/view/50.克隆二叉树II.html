题目描述

克隆一个带有随机节点指针的二叉树

特殊要求

  • 请使用以下数据结构 
class Node
    {
    int data;
    Node left;
    Node right
    Node random
    };

输入输出

输入

  • 第一行是n，表示二叉树的节点个数；当n=-1时

  • 之后的n行，每行是每个节点的左子树、右子树、随机节点的节点编号；编号-1表示空

输出

  • 针对每组数据，克隆从编号为0的节点开始的二叉树；然后按照随机节点、左子树、右子树的先根遍历顺序输出这个二叉树

示例输入

3
                1 2
                -1 -1
                -1 -1
                -1

示例输出

0 1 2

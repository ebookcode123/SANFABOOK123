题目描述

判断一堆链表中否有环

特殊要求

  • 请使用以下链表结构 

    class Node
    {
    int data;
    Node* next;
    };
  • 请额外使用O(1)的空间

输入输出

输入

  • 第一行是n，表示节点个数；当n=-1，表示输入结束

  • 第二行是n个整数，表示每一个节点所指向的节点的编号（-1表示为空）

输出

  • 针对每组数据，输出是否存在环

示例输入

4

1 2 3 0

5

1 2 －1 4 －1

-1

示例输出

1

0

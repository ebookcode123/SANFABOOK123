题目描述

请翻转一个有向图

特殊要求

  • 请使用以下图形结构 

class Node
{
    int data;
    LinkedList<Node> neighbors;
    };

输入输出

输入

  • 第一行是n，表示节点个数，节点编号从0到n-1；当n=-1时，表示输入结束

  • 之后的n行，每一行的第一个元素是m，表示对应的节点有几个邻居；之后的m个整数表示这些邻居的编号

输出

  • 针对每组输出，输出翻转后的图形结构。一共输出n行，每一行分别是从0到n-1的节点的邻居编号（从小到大排列）

示例输入

4
                0
                3 0 2 3
                0
                0

示例输出

1 1
                0
                1 1
                1 1

题目描述

判断一个数字是否是忧郁数字？

如果一个数字能够通过有限次忧郁变换成为0，则是忧郁数字。

忧郁变换是对一个数字的每一位的平方数求积。

例如：

对于68

  • 68 => 6^2 * 8^2= 2304

  • 2304 => 2^2 3^2 0^2 * 4^2 = 0

因此68是忧郁数字

输入输出

输入

  • 第一行是n；如果n为-1，表示输入结束

输出

  • 针对每组数据，输出是否是忧郁数字 1/0

示例输入

68
                1
                -1

示例输出

1
                0

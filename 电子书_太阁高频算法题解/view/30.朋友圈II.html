题目描述

有n个小伙伴，每个小伙伴都认识0个或多个小伙伴（如果A认识B，B不一定认识A）。如果A认识B，B认识C，则通过B的介绍，A也能认识C。如果A、B、C能够两两认识，那么他们能够形成一个朋友圈。请问这n个小伙伴能够形成几个互不相关的朋友圈。

输入输出

输入

  • 第一行是n；当n=-1，表示输入结束
  • 第二行是m，表示m个认识关系
  • 第三行是m对认识关系

输出

  • 针对每组输入，输出朋友圈的个数

示例输入

3
1
1 2
3
2
0 1 1 2
3
2
0 1 1 0
-1

示例输出

3
3
2

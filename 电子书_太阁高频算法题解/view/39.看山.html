题目描述

有n座山峰排成一排，站在一座山峰上向右看，映入眼帘的是右侧第一个更高的山峰。请依次找到这n个山峰的下一个高峰。

例如：


      |
  |   |
| |   |
| | | |
| | | |
3 4 2 5

我们分别看到的山峰的高度是：

  • 在第一个高峰，看到的是4
  • 在第二个高峰，看到的是5
  • 在第三个高峰，看到的是5
  • 在第四个高峰，什么都看不到，因此输出-1

输入输出

输入

  • 第一行是n；当n=-1时，表示输入结束

  • 第二行是n个山峰的高度

输出

  • 针对每组数据，依次输出这n个山峰向右看到的第一个山峰的高度

示例输入

4

3 4 2 5

-1

示例输出

4 5 5 -1

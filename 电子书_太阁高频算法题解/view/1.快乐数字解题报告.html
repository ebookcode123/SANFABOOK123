Iteration

不断的进行happy运算，直到出现1或者出现环。


                import java.io.*;
                import java.util.*;

                class Happiness{
                boolean isHappy(int n){
                HashSet<Integer> cache = new HashSet<Integer>();

                while( cache.add(n) ) n = transform(n);

                return n==1? true:false;
                }

                int transform(int n){
                int result = 0;
                while(n!=0){
                result += (n%10)*(n%10);
                n=n/10;
                }
                return result;
                }
                }

                public class Solution {
                public static void main(String[] args) throws FileNotFoundException{
                Scanner in = new Scanner(new File("input.txt"));

                int n = in.nextInt();
                Happiness happiness = new Happiness();
                while(n!=-1){
                System.out.println(happiness.isHappy(n));
                n = in.nextInt();
                }

                in.close();
                }
                }

常见BUG

  • 使用(n%10)^2来计算平方
  • 没有定义变量n
  • 使用重复的cache

Time complexity

输入一个n，我们并不知道需要计算多久。但是我们可以考虑一个happy变换会得到什么？

假设输入的是最大的正整数，有10位。经过happy变换，能够得到的最大的数是 10*9^2=810。 换句话说，所有的正整数经过变换，都会小于810。因此我们计算的复杂度一定在 O(810)，于是为O(1)

Recursion

我们每次得到一个n，都要重新计算，能否复用之前的结果呢？把计算过的n都保存了。

能否进一步加速呢？我们每次运算的中间结果也是有价值的，因此也保存下来。


                import java.io.*;
                import java.util.*;

                class Happiness{

                HashMap<Integer, Boolean> cache;

                Happiness(){
                cache = new HashMap<Integer, Boolean>();
                cache.put(1,true);
                }

                boolean isHappy(int n, int depth){
                if( cache.containsKey(n) ){
                System.out.println(depth);
                return cache.get(n);
                }
                cache.put(n,false);

                boolean result = isHappy(transform(n),depth+1);
                cache.replace(n, result);
                return result;
                }

                int transform(int n){
                int result = 0;
                while(n!=0){
                result += (n%10)*(n%10);
                n=n/10;
                }
                return result;
                }
                }

                public class Solution {
                public static void main(String[] args) throws FileNotFoundException{
                Scanner in = new Scanner(new File("input.txt"));

                int n = in.nextInt();
                Happiness happiness = new Happiness();
                while(n!=-1){
                System.out.println(happiness.isHappy(n, 1));
                n = in.nextInt();
                }

                in.close();
                }
                }

常见BUG

  • cache忘记赋值
  • cache首先设定为false，当找到最终结果时，需要在整个返回的路径上重新赋值（主要是处理true的情况）

如何节省空间

实际上，上述的cache算法有个潜在的风险，如果正整数判断多了，会占用大量的cache空间。 但是基于以上的算法复杂度分析，我们只需要保存810以内的结果即可。 但是对于每个输入的数，首先进行一次变换，让它进入810的范围呢。


                import java.io.*;
                import java.util.*;

                class Happiness{

                HashMap<Integer, Boolean> cache;

                Happiness(){
                cache = new HashMap<Integer, Boolean>();
                cache.put(1,true);
                }

                boolean isHappy(int n, int depth){
                if( cache.containsKey(n) ){
                System.out.println(depth);
                return cache.get(n);
                }
                cache.put(n,false);

                boolean result = isHappy(transform(n),depth+1);
                cache.replace(n, result);
                return result;
                }

                int transform(int n){
                int result = 0;
                while(n!=0){
                result += (n%10)*(n%10);
                n=n/10;
                }
                return result;
                }
                }

                public class Solution {
                public static void main(String[] args) throws FileNotFoundException{
                Scanner in = new Scanner(new File("input.txt"));

                Happiness happiness = new Happiness();
                for(int i=2;i<=810;++i){
                happiness.isHappy(i,1);
                }

                int n = in.nextInt();
                while(n!=-1){
                System.out.println(happiness.isHappy( happiness.transform(n), 1));
                n = in.nextInt();
                }

                in.close();
                }
                }

常见BUG

  • cache忘记初始化
  • cache赋错初始值
  • 输入n后，忘记先进行一次transform
